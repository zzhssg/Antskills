## 2. Prototype-Only（不要照搬）

| 维度 | Demo 行为 | 生产应替换为 |
|------|----------|-------------|
| 数据 | Mock 数据在 JS 里 | 后端 API 返回 |
| L2 计算 | 在浏览器 JS 执行 | 后端 Python 执行 |
| L3 LLM | 直接 fetch API（前端暴露 Key） | 后端代理调用 |
| 错误处理 | console.log / 静默失败 | ErrorCard + Retry |
| 刷新 | 手动点按钮 | 定时 + WebSocket |
| URL 路由 | 单页无路由 | 产品路由体系集成 |
| 认证 | 无 | 用户登录态 |
