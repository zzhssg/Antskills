# SeerClaw Smart Money Analyzer Skill

Standalone package for the **Analyzer** side of SeerClaw smart money: address analysis, wallet profile, radar, follow advice, positions, trade history, and per-coin chart analysis.

</div>

## What is inside?

- Analyzer-only `SKILL.md`
- Analyzer business / API / AI / viz references
- Analyzer AI payload templates
- Analyzer-only output validator script

## Use it for

- `/seerclaw-smart-money-analyzer analyze this wallet`
- `/seerclaw-smart-money-analyzer build the radar and follow-advice cards`
- `/seerclaw-smart-money-analyzer fix chart coin switching and trades table`

## Do not use it for

- coin + time-window scan flow
- Scanner address list rendering
- Scanner consensus generation


## Hi-fi reference

- `references/analyzer-hi-fi.html`: Analyzer 高保真 HTML 参考页，默认直出 Analyzer 成品态 UI。
