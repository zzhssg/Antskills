# SeerClaw 聪明钱 Analyzer Skill

这是一个只包含 **Analyzer** 链路的独立包：负责地址分析、画像与 Radar、跟单建议、持仓表、交易历史，以及按币切换的图表分析。

</div>

## 包内内容

- Analyzer 专属 `SKILL.md`
- Analyzer 的 business / API / AI / 可视化参考文件
- Analyzer AI 输入输出样例
- Analyzer 专属校验脚本

## 适用场景

- `/seerclaw-smart-money-analyzer 分析这个地址值不值得跟`
- `/seerclaw-smart-money-analyzer 实现 Radar 和跟单建议卡`
- `/seerclaw-smart-money-analyzer 修复切币图表和交易表`

## 不适用场景

- 按币种和时间窗扫描地址
- Scanner 地址卡列表
- Scanner 共识文案
