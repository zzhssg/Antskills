Standalone package for the **Scanner** side of SeerClaw smart money: scan by coin and time window, render smart-money cards, generate consensus copy, and hand off to Analyzer.

- Scanner-only `SKILL.md`
- Scanner business / API / AI / viz references
- Scanner AI payload templates
- Scanner-only output validator script

## Use it for

- `/seerclaw-smart-money-scanner build the BTC scan flow`
- `/seerclaw-smart-money-scanner fix Scanner loading state and chart markers`
- `/seerclaw-smart-money-scanner wire scan endpoint and consensus copy`

## Do not use it for

- wallet profile pages
- radar score rendering
- follow-advice logic
- trade-history analyzer work
